### 1. 기반 UI 시스템 (`static/js/ui/components.js`)
```javascript

```

**주요 기능:**
- 모달 시스템 (showModal, hideModal)
- 토스트 알림 (showToast, hideToast)
- 로딩 오버레이 (showLoading, hideLoading)
- 재사용 가능한 UI 컴포넌트
- 애니메이션 및 전환 효과
